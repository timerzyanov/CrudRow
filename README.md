# Getting Started
Use:
 - @RunWith(SpringRunner.class)
 - @SpringBootTest
 - @Test
 - @SpringBootApplication
 - @Autowired
 - @Route
 - @Query
 - @Entity
 - @Data
 - @SpringComponent
 - @UIScope

How to use:
1. Download this project
2. Run mnv clean install
3. Run main class as application
